# Guia Simples: Como Usar IA para Criar Páginas Web

## 🎯 **O Básico: Como "Falar" com a IA**

Pense na IA como um assistente muito inteligente, mas que precisa de instruções claras. É como pedir ajuda para alguém - quanto mais específico você for, melhor será o resultado.

## 📝 **3 Regras de Ouro**

### 1. **Seja Específico**
❌ *"Crie uma página"*
✅ *"Crie uma página de vendas para meu curso de culinária, com botão para WhatsApp e cores azul e branco"*

### 2. **Dê Contexto**
❌ *"Faça um site bonito"*
✅ *"Sou dentista e preciso de um site para atrair pacientes em Florianópolis. Quero mostrar meus serviços e facilitar agendamentos"*

### 3. **Peça Exemplos**
❌ *"Como fazer isso?"*
✅ *"Me mostre passo a passo como criar um botão de contato que abra o WhatsApp"*

## 🗣️ **Fórmula Simples para Prompts**

```
"Você é um [PAPEL]. 
Preciso de [O QUE] para [FINALIDADE].
O público-alvo é [QUEM].
Deve ter [CARACTERÍSTICAS ESPECÍFICAS]."
```

**Exemplo prático:**
```
"Você é um criador de sites.
Preciso de uma página inicial para minha loja de roupas.
O público-alvo são mulheres de 25-45 anos.
Deve ter cores femininas, fotos dos produtos e botão para Instagram."
```

## 💡 **Dicas Práticas**

### **Para Começar:**
- Use frases como: *"Explique como se eu fosse iniciante"*
- Peça: *"Me dê o código pronto para copiar e colar"*
- Sempre pergunte: *"Como faço isso funcionar?"*

### **Para Melhorar:**
- Diga: *"Torne isso mais bonito"*
- Peça: *"Deixe responsivo para celular"*
- Solicite: *"Adicione mais cores/animações"*

### **Para Resolver Problemas:**
- Informe: *"Não está funcionando, o que fazer?"*
- Explique: *"Quero mudar apenas [parte específica]"*
- Copie e cole o erro que apareceu

## 📱 **Exemplos Reais para Diferentes Negócios**

### **Restaurante:**
*"Crie uma página simples para meu restaurante. Preciso mostrar o cardápio, localização e telefone para delivery. Cores quentes e fotos apetitosas."*

### **Serviços (salão, oficina, etc.):**
*"Faça um site para meu salão de beleza. Quero mostrar os serviços, preços e facilitar agendamento pelo WhatsApp. Design elegante e feminino."*

### **Loja Online:**
*"Preciso de uma página de vendas para meu produto [nome do produto]. Deve convencer a comprar, mostrar benefícios e ter botão de pagamento."*

## ⚠️ **Cuidados Importantes**

- **Sempre teste** se funciona no celular
- **Verifique** se os links/botões funcionam
- **Peça ajuda** se algo der erro
- **Comece simples** - você pode melhorar depois

## 🔄 **Como Melhorar aos Poucos**

1. **Primeira versão:** Peça algo básico e funcional
2. **Segunda versão:** "Torne mais bonito"
3. **Terceira versão:** "Adicione [funcionalidade específica]"
4. **Sempre:** "Explique o que mudou"

## 💬 **Linguagem Simples**

Em vez de termos técnicos, use:
- "Página" em vez de "website"
- "Botão" em vez de "elemento"
- "Cor" em vez de "paleta"
- "Funcionar" em vez de "implementar"

**Lembre-se:** A IA está aí para ajudar! Não tenha medo de fazer perguntas simples ou pedir explicações. Quanto mais você usar, mais fácil fica.
